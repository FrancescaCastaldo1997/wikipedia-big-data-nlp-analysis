# Wikipedia Big Data & NLP Analysis

This project focuses on large-scale analysis and automatic classification of Wikipedia articles using Big Data technologies and Natural Language Processing (NLP) techniques.

The project combines exploratory data analysis (EDA), distributed data processing with Spark, and machine learning models for text classification.

---

## Project Goals

- Analyze Wikipedia article categories
- Perform exploratory data analysis (EDA)
- Identify linguistic trends across categories
- Build an automatic text classification model
- Process large datasets using Spark DataFrames

---

## Technologies Used

- Python
- PySpark
- Pandas
- Machine Learning
- Natural Language Processing (NLP)
- Google Colab
- Big Data concepts

---

## Analysis Performed

The project includes:

- Article count by category
- Average article length
- Longest and shortest articles
- Word cloud generation
- Text preprocessing
- Automatic category classification

---

## Machine Learning Tasks

- Text classification
- NLP preprocessing
- Feature extraction
- Model training and evaluation

---

## Big Data Workflow

The dataset was processed using distributed computing concepts with Spark DataFrames.

Workflow:

1. Load dataset from cloud storage
2. Convert Pandas DataFrame into Spark DataFrame
3. Perform large-scale text analysis
4. Train and evaluate classification models

- Automatic article categorization

- Editorial trend analysis
- Large-scale text analytics

Wikipedia dataset containing:
